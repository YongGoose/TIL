# CHAPTER 01. 머신러닝과 딥러닝의 개념

## 규칙 기반 AI와 머신러닝

- **규칙 기반 AI**: 사람이 직접 규칙(if-else)을 정의하여 동작하는 방식
  - 예: 스팸 필터에서 특정 키워드가 포함되면 스팸으로 분류
  - 한계: 규칙이 많아질수록 유지보수가 어렵고, 예외 상황에 대응하기 힘듦
- **머신러닝**: 데이터를 통해 패턴을 스스로 학습하는 방식
  - 명시적인 규칙 없이도 데이터에서 규칙을 발견
  - 데이터가 많을수록 성능이 향상됨

## 전통적인 머신러닝과 딥러닝의 차이점

- **전통적인 머신러닝**
  - 사람이 직접 특징(feature)을 설계하고 추출해야 함 (Feature Engineering)
  - 비교적 적은 데이터로도 학습 가능
  - 알고리즘 예시: 선형 회귀, 결정 트리, SVM 등
- **딥러닝**
  - 신경망이 스스로 특징을 학습함 (자동 특징 추출)
  - 대량의 데이터와 높은 컴퓨팅 파워가 필요
  - 이미지, 음성, 자연어 등 비정형 데이터에서 뛰어난 성능
  - 알고리즘 예시: CNN, RNN, Transformer 등
