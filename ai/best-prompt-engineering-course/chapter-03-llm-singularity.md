# CHAPTER 03. LLM, 특이점의 시작

## GPT-4의 뛰어난 성능

- **GPT-4**: OpenAI가 발표한 멀티모달 대규모 언어 모델
  - 텍스트뿐 아니라 이미지 입력도 처리 가능
  - 변호사 시험, 의사 자격시험 등 전문 시험에서 상위권 성적을 기록
- 이전 모델(GPT-3.5) 대비 추론 능력, 정확성, 안전성이 크게 향상
- **AGI(Artificial General Intelligence)** 논의가 본격적으로 시작되는 계기
  - 단순 언어 생성을 넘어 복잡한 문제 해결 능력을 보여줌

## 소프트웨어 개발 방식의 진화

- **기존 개발 방식**: 개발자가 직접 로직을 코드로 작성하여 문제를 해결
- **LLM 시대의 개발 방식**: 자연어로 요구사항을 전달하면 AI가 코드를 생성하거나 직접 태스크를 수행
- 소프트웨어 개발의 패러다임 변화
  - 코드 중심 → **프롬프트 중심**으로 전환
  - 개발자의 역할이 코드 작성자에서 AI 협업자로 확장
- **No-Code/Low-Code** 도구와 LLM의 결합으로 비개발자도 소프트웨어 제작이 가능해짐

## 새로운 기회 찾기

- LLM의 등장으로 기존에 불가능했던 서비스와 비즈니스 모델이 가능해짐
- **자동화 가능 영역의 확대**: 고객 응대, 문서 작성, 데이터 분석 등 반복적 지식 노동의 자동화
- 산업별 LLM 활용 사례
  - 의료: 의료 기록 요약, 진단 보조
  - 법률: 판례 검색, 계약서 검토
  - 교육: 개인 맞춤형 튜터링, 자동 채점
  - 마케팅: 콘텐츠 생성, 고객 세그먼트 분석

## 새로운 기회의 시작

- LLM 기반 스타트업과 서비스가 폭발적으로 성장 중
- **프롬프트 엔지니어링**이 핵심 역량으로 부상
  - AI를 잘 활용하는 능력이 곧 경쟁력
  - 프롬프트 하나로 결과물의 품질이 크게 달라짐
- AI 시대에서의 경쟁력 = **문제를 정의하는 능력** + **AI와 효과적으로 소통하는 능력**
- 기술적 진입 장벽이 낮아진 만큼, 도메인 전문성과 창의성이 더 중요해짐
